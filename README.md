# MovieMemoirs

My second own project.

It's an app, where the user can search for the title of the movie he is interested in via a network call to the OMDb API. 
After entering the title, a list of films will appear. After tapping on any of the titles, the user will see details about the selected movie. 
On the movie details screen, the user can add the movie to the list of their favorite movies. In turn, the user can share the list of his favorite movies with anyone.

## What's inside?

- MVVM
- 100 % Programmatic UI
- NetworkManager
- Protocols
- JSON
- URLSession
- Network Call
- UITabBarController
- UICollectionViewController
- UICollectionView
- UIActivityViewController
- Extensions
- Unit testing
